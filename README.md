# Sports-image-classification
Deep Learning model for classifying sports images (Baseball, Basketball, Soccer etc)

##  Overview
This project classifies sports images into:
- Baseball
- Basketball
- Soccer etc

##  Models Used
- CNN (Baseline)
- MobileNetV2 (Transfer Learning)

## Accuracy
- CNN: 98%
- MobileNetV2: 99%

##  Tech Stack
- Python
- TensorFlow / Keras

##  Dataset
Dataset not included due to size.
Use: [Kaggle Dataset Link: https://www.kaggle.com/datasets/gpiosenka/sports-classification]
trained model link: https://www.kaggle.com/datasets/gpiosenka/sports-classification?select=EfficientNetB0-100-%28224+X+224%29-+98.40.h5

## How to Run
pip install -r requirements.txt
python app.py
