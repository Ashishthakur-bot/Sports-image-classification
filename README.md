# Sports-image-classification
Deep Learning model for classifying sports images (Baseball, Basketball, Soccer etc)

## 📌 Overview
This project classifies sports images into:
- Baseball
- Basketball
- Soccer etc

## 🧠 Models Used
- CNN (Baseline)
- MobileNetV2 (Transfer Learning)

## 📊 Accuracy
- CNN: 98%
- MobileNetV2: 99%

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras

## 📂 Dataset
Dataset not included due to size.
Use: [Kaggle Dataset Link: https://www.kaggle.com/datasets/gpiosenka/sports-classification]

## ▶️ How to Run
pip install -r requirements.txt
python app.py
